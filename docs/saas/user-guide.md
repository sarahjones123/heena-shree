## Overview

TaskFlow is a cloud-based project management platform supporting task tracking and collaboration.

---

## Creating a Project

1. Navigate to Dashboard.
2. Select **New Project**.
3. Enter project details.
4. Click **Create**.

---

## Task Lifecycle States

- To Do
- In Progress
- Completed

Tasks can be updated via drag-and-drop or status dropdown selection.

---

## Reporting

Users can generate:

- Task completion reports
- Workload distribution reports
- Deadline analysis reports

Reports are exportable in PDF and Excel formats.
