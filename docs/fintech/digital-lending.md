# Digital Lending System – User Guide

## Loan Application Workflow

1. Create new application.
2. Enter applicant details.
3. Upload documents.
4. Submit for verification.

---

## Approval Matrix

| Loan Amount | Approval Authority |
|------------|-------------------|
| Up to 5 Lakhs | Loan Manager |
| 5–20 Lakhs | Regional Head |
| Above 20 Lakhs | Credit Committee |

- Timestamped updates
